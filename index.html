Short and Easy Unity Cross Platform Integration Guide

Windows/Android/iOS/Other Unity - I personally integrated a WebGL application.

(Integration documentation for other platforms can be obtained after logging into their website)

1: Request an application key by adding your application to the Panel.

Register and log in to PacketSDK, go to the Panel to add your application and obtain an application key. The "appkey" is a critical way of tracking earnings, ensure it is correctly set in subsequent integration steps.

2: Download the Unity Cross Platform integration package.

Download the PacketSDK Unity-CrossPlatform integration package packet_sdk_unity_crossplatform-1.0.*.zip and unpack it.

After unpacking the package, you will see the following contents:

  • PacketSDK.dll - Library required for Unity integration
  • README.html

3: Copy the required file to your Unity project.

Copy PacketSDK.dll to the "Assets" directory of your project.

4. Create a script.

E.g., callscript.cs, create a new object - Create Empty (name it e.g., CallScript), add the callscript.cs script to the newly created CallScript object.

Insert your "appkey" into the script, which you received after logging into the platform and creating an application in the Panel.


using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using PacketSDKAPI;

public class callscript : MonoBehaviour
{
    static void onPacketSDKCallBack(packet_status_t status)
    {
        Debug.Log("Packet SDK callback with status: " + PacketSDK.StatusToString(status));
    }

    // Start is called before the first frame update
    void Start()
    {
        Debug.Log("MyScript started!");

        PacketSDK.SetCallBack(onPacketSDKCallBack);

        packet_status_t status = PacketSDK.SetAppKey("appkey");
        Debug.Log("SetAppkey: " + PacketSDK.StatusToString(status));
        status = PacketSDK.Start();
        Debug.Log("SDKStart: " + PacketSDK.StatusToString(status));
    }

    // Update is called once per frame
    void Update()
    {
        
    }
}

5. Select the target platform in Build Settings and build your application.

Device data and revenues will become visible in the Packet SDK panel within 24 hours.

Detailed integration documentation for Unity Cross Platform and other platforms can be obtained after logging in.
